# n8n Automation Templates

A collection of [n8n](https://n8n.io) automation templates for various platforms and services.

---

## Getting Started

1. **Install n8n**: Follow the [official n8n documentation](https://docs.n8n.io/hosting/) for installation options (Docker, npm, or n8n Cloud).

2. **Import a template**: 
   - Open n8n and go to **Workflows**
   - Click **Import from File**
   - Select any `.json` file from this repository

3. **Configure credentials**: Each workflow may require API keys or credentials for the services it connects to.
